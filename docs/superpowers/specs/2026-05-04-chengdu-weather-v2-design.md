# 成都天气应用 V2 — 设计规格

**日期：** 2026-05-04
**状态：** 已批准
**技术栈：** React 18 + Vite + Tailwind CSS

---

## 1. 项目概述

在 V1 基础上进行视觉升级和功能扩展：更真实的毛玻璃效果、现代动效、重庆天气支持、完整手机适配。

---

## 2. 视觉设计

### 2.1 毛玻璃效果增强

| 层级 | 属性 | 值 |
|------|------|-----|
| 背景层 | backdrop-blur | blur(8px) |
| 内容层 | backdrop-blur | blur(20px) |
| 卡片背景 | bg-white/65 | rgba(255,255,255,0.65) |
| 边框 | border-white/40 | 1px 细边框 |
| 内阴影 | box-shadow | inset 0 1px 1px rgba(255,255,255,0.5) |

### 2.2 配色方案

| 角色 | Hex | 用途 |
|------|-----|------|
| Background | 渐变 `#F0F9FF → #E0F2FE → #BAE6FD` | 页面全屏背景 |
| Primary | #0891B2 | 强调色 |
| Text Dark | #0C4A6E | 主要文字 |
| Text Muted | #64748B | 次要文字 |
| Card Shadow | 0 8px 30px rgba(8,145,178,0.15) | 卡片阴影 |

### 2.3 动效

| 场景 | 效果 | 时长 |
|------|------|------|
| 主卡片入场 | fadeIn + scale(0.95→1) + blur 出现 | 500ms |
| 预报卡片 | staggered delay 80ms | 400ms |
| 卡片 hover | scale(1.02) + 阴影扩散 | 200ms |
| 温度数字 | 数字跳动 | 600ms |
| 背景光晕 | 缓慢移动渐变 | 20s 循环 |

## 4. 响应式适配

### 断点

- < 375px：单列布局，字体缩放
- 375px - 767px：手机布局
- 768px - 1023px：平板布局
- ≥ 1024px：桌面布局

### 手机优化

- 最小触摸区域 44px
- 卡片圆角调整（手机 16px，桌面 24px）
- 底部安全区域适配
- 横向滚动预报条优化
