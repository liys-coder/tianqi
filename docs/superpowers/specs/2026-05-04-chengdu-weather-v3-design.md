# 成都天气应用 V3 — 视觉与功能升级设计

## 概述

升级天气应用为现代极简风格，增强毛玻璃效果，添加丰富动效，增加详细数据指标。

## 视觉风格

### 整体风格
- **现代极简** — 大面积留白、精致细线、微动画
- **轻盈毛玻璃** — 高透明度 (opacity 0.4-0.5)、强模糊 (blur 24-32px)、空气感

### 配色方案
- 背景：极淡蓝白渐变 (#F8FAFC → #E0F2FE)，微动态流动
- 卡片：rgba(255,255,255,0.4-0.5) + blur(24-32px)
- 文字：主色 #0F172A (slate-900)，次色 #64748B (slate-500)
- 强调色：#0284C7 (sky-600)

### 字体
- 主字体：Inter
- 数字：等宽字体 (tabular-nums)

## 页面结构

### 1. 顶部城市切换器
- 位置：页面顶部
- 样式：横向排列的胶囊按钮
- 交互：当前城市高亮 (实色背景)，点击切换城市

### 2. 主天气卡片 (HeroCard)
- 内容：温度大字 (72px)、天气描述、体感温度、天气图标
- 动效：入场 fade-in-scale + 上浮

### 3. 基础指标条 (MetricsBar)
- 内容：湿度、风速、体感温度
- 布局：横向紧凑排列，图标 + 数值

### 4. 5日预报条 (ForecastStrip)
- 内容：日期 + 图标 + 最高/最低温
- 布局：横向滚动 (mobile) / 网格 (desktop)
- 动效：stagger 入场

### 5. 详细指标网格 (新增)
- 布局：2x3 网格卡片
- 内容：
  - 紫外线指数 (UV Index)
  - 能见度 (Visibility)
  - 气压 (Pressure)
  - 日出时间 (Sunrise)
  - 日落时间 (Sunset)
  - 云量 (Cloud Cover)
- 交互：可滚动查看

## 动效设计

### 页面入场
- 卡片依次淡入上浮，stagger 延迟 100ms
- 数字滚动效果 (count-up)

### 交互反馈
- 悬停：卡片上浮 4px + 阴影加深
- 点击：轻微缩放 (scale 0.98)

### 切换过渡
- 城市切换：内容 crossfade 过渡
- 数据更新：数值平滑过渡

## API 扩展

需要从 Open-Meteo 获取额外参数：
- `daily=uv_index_max,sunrise,sunset,visibility_mean,cloud_cover_mean,surface_pressure_mean`

## 验收标准

1. 页面在移动端和桌面端均美观
2. 毛玻璃效果轻盈通透
3. 动效流畅优雅
4. 详细指标可滚动查看
5. 构建无错误
